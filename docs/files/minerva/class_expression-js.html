signature

bbopx.minerva.class_expression.prototype.signature = function()

A cheap way of identifying if two class_expressions are the same.  This essentially returns a string of the main attributes of a type.  It is meant to be semi-unique and collide with dupe inferences.

BUG/WARNING: At this point, colliding signatures should mean a dupe, but non-colliding signamtes does not guarantee that they are not dupes (think different intersection orderings).
